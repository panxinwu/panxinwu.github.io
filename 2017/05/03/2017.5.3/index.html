#学习笔记# 知乎live《高度自律：为了高度自由》

情绪管理

  1. 每一场痛苦背后，都有不理性的活该
  2. 气质冷静，不该在动脑子的时候动感情
  3. 减少依赖性，接受多样性
  4. 谁的青春不迷茫，人都差不多，遇到的问题也差不多，人具有共性，都有自己的问题

    正确的三观

社会很残酷，有的事实可以摧毁你的三观，但是要坚持自己的正确的三观

  1. 真诚坦荡，拒绝装逼。拒绝装逼，不要装作很懂，装逼遭雷劈。
  2. 世界太喧嚣，不要在意别人的嘴碎、看法。有自己的主见。内心坚强，不要太在意别人说什么。不利于自己的发展和调整
  3. 不盲目自信，不妄自菲薄。有些东西没有想象的难，敢于尝试，自己不知道的时候不要盲目自信。当别人都会得东西，要慢慢接触不要妄自菲薄。自己也可以的。
  4. 时间换金钱，专业换尊重。专业水平没达到钱，先低调，提升自己，最终会得到别人的专业尊重
  5. 靠谱是最低成本的社交方式。拒绝拍马屁，首先是靠谱的人，才能跟别人真心交朋友、合作等社交方式

“不自律”的恶性循环

被一些空虚的东西填满自己。很多因素交叉导致的：

  1. 羡慕别人，高达威猛帅，升职加薪
  2. 自己胆小不帅气急败坏停滞不前
  3. 嫉妒别人，肤白貌美，高贵干练独当一面
  4. 自己面黄赘肉一无所长，自卑懦弱颓废抱怨

    励志鸡汤为何收获甚微

  5. 少喝鸡汤，鸡血粉丝团，几日训练营等以盈利为目的少参加。
  6. 大牛的奋斗历程，成功学教材，少听

    普通人怎样寻求进步

  7. 观摩活法、研究自己、按自己的意愿活一生，发现意义、踏上征程、循环往复。
  8. 经历不可复制，但能受到启发，从比自己稍好一些的那里得到一些启发

与其艳羡那个人，不如成为那个人，自己发光，美丽，具备价值，才是正事。

接受平淡现实

  1. 没什么精心动魄，多的是平凡得琐碎。任何的工作，大部分都枯燥无聊。
  2. 精良并非朝夕，而是长久坚持。精英们精气神都比较好，体态比较好，保持运动，使自己时时保持爽朗的状态。小事坚持反应的是坚持的状态。
  3. 恒毅力，持续的热情和付出

    自律是自我选择

  4. 不被强迫，纯属自愿
  5. 自己选的路，跪着也要走完
  6. 不愿意相亲，耐得住等真爱寂寞
  7. 奋斗事业，忍的了长辈催婚催娃

分类计划（已经列入自己的代办清单）

  1. 重要且紧急
  2. 紧急不重要
  3. 重要不紧急
  4. 不重要且不紧急

时间管理摒弃死板、学会变通

  1. 工作不确定性强
  2. 建立及时调整计划意识
  3. 保持工作性社交各种平衡

善于使用工具

  1. 印象笔记
  2. 马克飞象

仪式感很重要，每天对自己有个总结

避免干扰因素出现

  1. 关闭手机不必要的通知-保持电话微信等，微博知乎等推送关掉
  2. 固定时间消遣- 打游戏、刷社交、看直播。这些习惯没有好坏之分，不能没有选择性的来做这些事情。固定时间做固定的事情。及不重要也不紧急的事情放在固定时间点做，放松。

拒绝缺乏思考的拼命勤奋

  1. 从根本避免在错误规划的道路上越走越远
  2. 精力守恒，避免三分钟热度，虎头蛇尾
  3. 更改碎片化阅读习惯，随身携带书纸笔

金钱自律法则

  1. 钱是赚出来的，不是省出来的
  2. 努力赚钱不俗，打工赚钱不low
  3. 将知识与爱好变现是绝对快了
  4. 不是买不起贵的，而是只买对的

用自律培养自己真正的爱好

  1. 休闲消遣不是爱好。看电视、打游戏不是爱好，一技之长才是真正的爱好。有一技之长的人一辈子都会比别人更丰富一些
  2. 爱好不能浅尝辄止。一技之长要常年坚持
  3. get任何技能，一些小技能感觉很简单，做起来其实蛮有意思，很快就可以get到一些小技能很有意思

健身、保养、约会、旅行都是自律

  1. 我要开挂工作、也要精彩生活，工作之外的时间才是拉开人与人之间差距的时间。除了工作之外，哪些东西拿出来是根别人不一样的点
  2. 德智体美劳全面发展并不可笑。多全面发展。多方面让自己变的更好
  3. 打铁还需自身硬，健康是万物之本。多锻炼身体！
  4. 保养非女生专利，男生也是必须。

    进入自己的正向循环 负向循环：尝试-挫败-气馁-停止尝试-更加挫败

  5. 只会读书的圣贤，无法融入社会。多思考
  6. 人际关系要维持，多反省少抱怨。
  7. 旅行： 学会慢活，才能快活
